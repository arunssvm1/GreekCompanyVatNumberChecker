Sample: vatchecker

--------
Summary:
--------

This project exploits the web service provided by www.gsis.gr. It is a simple
android application that operates on systems 1.6 and above. 
The user is forced to insert the VAT number of a company. The application presents the
company information.

--------
Details:
--------

The application starts with a simple activity (VatCheckerActivity).
A form will give the opportunity to the user to insert only numerical values.
The form checks if the user gives VAT numbers larger than 9 digits.
If the VAT is correct the information is presented elsewhere a short
notification ,with an error text, is presented.
