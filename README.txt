Greek-Company-Vat-Number-Checker

--------
Summary:
--------

This sample project uses the web service provided by http://www.gsis.gr.
The user is able to insert the VAT number of a company. The application checks the existence 
of the company and presents the company information to the user.

-----------------
Operating System:
-----------------

Google Android 1.6+

--------
Details:
--------

The application starts with a simple activity (VatCheckerActivity).
A form will give the opportunity to the user to insert the VAT number of 
the company (only numerical values & length equal to 9 digits).

If the VAT is correct, the company information is presented, else a short
error notification is presented to the user.
